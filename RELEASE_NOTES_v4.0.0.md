# CACAYO Member System v4.0.0

White-label UI redesign for tablet-friendly staff operations and a simpler customer portal.

## Staff
- Dominant CACAYO logo and no visible Cash to Dine branding.
- Three primary menus: Transaksi, History Transaksi, Daftar Member.
- Transaction page separates Gunakan Saldo and Top Up.
- Daily grouped report with Print / Export PDF.
- Searchable member list and member transaction detail.
- Promo image manager: automatic 1200 × 800 crop, stored securely through RPC.

## Customer
- Simplified login and balance dashboard.
- Touch PIN keypad for balance approval.
- After approval, customer is automatically shown the active promo image.
